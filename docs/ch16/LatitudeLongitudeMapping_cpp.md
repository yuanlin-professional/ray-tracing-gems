# LatitudeLongitudeMapping.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 经纬度映射 / 均匀球面采样 (Latitude-Longitude Mapping / Uniform Sphere Sampling) |
| **用途** | 在半径为 $R$ 的球面 (sphere) 上均匀采样一个点 |
| **应用场景** | 均匀球面采样、环境贴图 (environment map) 的经纬度参数化、全向光源的方向采样 |

---

## 2. 数学推导

### 2.1 球面均匀 PDF

球面面积为 $4\pi R^2$，均匀采样的 PDF 为：

$$
f(\omega) = \frac{1}{4\pi R^2}
$$

### 2.2 球坐标参数化

球坐标 $(\theta, \phi)$，其中 $\theta \in [0, \pi]$ 为天顶角，$\phi \in [0, 2\pi)$ 为方位角。

面积元素 $dA = R^2 \sin\theta\, d\theta\, d\phi$。

### 2.3 Archimedes 等面积投影

利用换元 $a = \cos\theta$，令 $a$ 在 $[-1, 1]$ 上均匀分布：

$$
a = 1 - 2u_0 \quad \Rightarrow \quad \cos\theta = 1 - 2u_0
$$

$$
\sin\theta = \sqrt{1 - a^2} = b
$$

$$
\phi = 2\pi u_1
$$

### 2.4 直角坐标

$$
x = R \cdot b \cdot \cos\phi, \quad y = R \cdot b \cdot \sin\phi, \quad z = R \cdot a
$$

这等价于 Archimedes 定理：球面与其外接圆柱之间的等面积投影。

---

## 3. 完整代码与逐行注释

```cpp
a = 1 - 2*u[0];         // cosθ 在 [-1,1] 上均匀分布
                          // u[0]=0 → a=1（北极），u[0]=1 → a=-1（南极）
b = sqrt(1 - a*a);       // sinθ = sqrt(1 - cos²θ)

phi = 2*M_PI*u[1];       // 方位角 φ 在 [0, 2π) 上均匀分布

x = R*b*cos(phi);        // 球面上点的 x 坐标
y = R*b*sin(phi);        // 球面上点的 y 坐标
z = R*a;                 // 球面上点的 z 坐标
```

---

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u[0]` | `float` | $[0, 1]$ | 均匀随机变量，控制天顶角（纬度） |
| `u[1]` | `float` | $[0, 1)$ | 均匀随机变量，控制方位角（经度） |
| `R` | `float` | $(0, +\infty)$ | 球面半径 |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| `x` | `float` | 球面上采样点的 x 坐标 |
| `y` | `float` | 球面上采样点的 y 坐标 |
| `z` | `float` | 球面上采样点的 z 坐标 |

---

## 5. 应用场景

- **全向光源**：点光源或球形光源的方向均匀采样
- **环境贴图采样**：作为基础均匀采样，配合重要性采样权重使用
- **蒙特卡洛积分**：球面上函数的数值积分（如球谐函数 (spherical harmonics) 系数计算）
- **粒子系统**：在球面上均匀生成粒子初始位置
- **碰撞检测**：在球面上均匀采样方向进行随机化碰撞测试

## 6. 与相关变换的对比

| 映射方法 | 目标域 | 分布 | 特点 |
|---------|--------|------|------|
| **经纬度映射 (本方法)** | 球面 | 均匀 | 最基本的球面采样；极点处采样点不聚集（因为使用了 $\cos\theta$ 均匀分布） |
| **圆锥采样** (`ConeSampling.cpp`) | 球冠 (cap) | 均匀 | 球面采样的子集，限制在锥角内 |
| **八面体同心映射** (`OctahedralConcentricMapping.cpp`) | 球面 | 均匀 | 无分支，对低差异序列 (low-discrepancy sequence) 更友好 |
| **朴素经纬度参数化** | 球面 | 非均匀 | 若直接用 $\theta = \pi u_0$ 则极点附近采样过密 |

> **注意**：本方法利用了 Archimedes 等面积投影的原理，通过在 $\cos\theta$（而非 $\theta$ 本身）上均匀采样来保证球面均匀性。这避免了朴素经纬度参数化中极点聚集的问题。
