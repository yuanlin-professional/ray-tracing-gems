# TriangleQuadFlipping.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 三角形采样——四边形翻折法 (Triangle Sampling via Quad Flipping) |
| **用途** | 将正方形 $[0,1]^2$ 上的均匀采样点映射到三角形内部，使用简单的"翻折"技巧确保均匀分布 |
| **应用场景** | 三角形面光源采样、三角形网格上的均匀采样、光子映射中三角形上的光子发射 |

---

## 2. 数学推导

### 2.1 基本思想

正方形 $[0,1]^2$ 的面积是三角形 $\{(\alpha,\beta) : \alpha+\beta \le 1, \alpha,\beta \ge 0\}$ 的两倍。将正方形对角线以上的点翻折到对角线以下，即可实现均匀映射。

### 2.2 翻折操作

给定 $(u_0, u_1) \in [0,1]^2$：

$$
\alpha = u_0, \quad \beta = u_1
$$

若 $\alpha + \beta > 1$（点在对角线以上）：

$$
\alpha' = 1 - \alpha, \quad \beta' = 1 - \beta
$$

此时 $\alpha' + \beta' = 2 - (\alpha+\beta) < 1$，点被翻折到三角形内部。

### 2.3 重心坐标

$$
\gamma = 1 - \alpha - \beta
$$

三角形上的点：

$$
P = \alpha P_0 + \beta P_1 + \gamma P_2
$$

### 2.4 均匀性证明

翻折操作是关于点 $(0.5, 0.5)$ 的中心对称。正方形上均匀分布经翻折后，三角形内每个点被覆盖恰好两次（原始点和翻折点），因此保持均匀性。

## 3. 完整代码与逐行注释

```cpp
alpha = u[0];                    // 初始重心坐标 α = u0
beta = u[1];                     // 初始重心坐标 β = u1

if (alpha + beta > 1) {          // 若 (α, β) 在对角线以上（不在三角形内）
    alpha = 1-alpha;              // 翻折：α' = 1 - α
    beta = 1-beta;                // 翻折：β' = 1 - β
    // 翻折后 α' + β' = 2 - (α+β) < 1，保证在三角形内
}

gamma = 1-beta-alpha;            // 第三个重心坐标 γ = 1 - α - β
                                  // 满足 α + β + γ = 1

P = alpha*P0 + beta*P1 + gamma*P2;
// 利用重心坐标 (barycentric coordinates) 计算三角形内的点
// P = α*P0 + β*P1 + γ*P2
```

---

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u[0]` | `float` | $[0, 1]$ | 均匀随机变量 |
| `u[1]` | `float` | $[0, 1]$ | 均匀随机变量 |
| `P0, P1, P2` | `vec3` | — | 三角形的三个顶点 |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| `alpha, beta, gamma` | `float` | 重心坐标，满足 $\alpha+\beta+\gamma=1$，$\alpha,\beta,\gamma \ge 0$ |
| `P` | `vec3` | 三角形内的均匀采样点 |

---

## 5. 应用场景

- **三角形面光源**：在三角形面光源上均匀采样发光点
- **面积光照**：蒙特卡洛直接光照中，在三角形光源表面上采样
- **光子发射**：光子映射中从三角形光源上均匀发射光子
- **表面散布**：在三角形网格表面上均匀散布粒子或纹理样本

## 6. 与相关变换的对比

| 方法 | 实现 | 保持分层性 | 均匀性 | 计算开销 |
|------|------|-----------|--------|---------|
| **四边形翻折法 (本方法)** | 简单（一次比较+可能的翻折） | 部分——翻折会破坏部分分层结构 | 均匀 | 最低 |
| **四边形扭曲法** (`TriangleQuadWarping.cpp`) | 简单（一次 sqrt + 乘法） | 更好——连续映射 | 均匀 | 稍高（一次 sqrt） |
| **Heitz 方法** | 较复杂 | 最好 | 均匀 | 中等 |

> **分层采样的问题**：翻折法的主要缺点是对角线处的不连续性——原本分层的采样点经翻折后可能产生重叠和间隙。如果使用分层采样或低差异序列，扭曲法 (`TriangleQuadWarping.cpp`) 通常是更好的选择。
