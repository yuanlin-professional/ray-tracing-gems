# SampleBilinearInterpolation.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 双线性插值采样 (Bilinear Interpolation Sampling) |
| **用途** | 在单位正方形 $[0,1]^2$ 上按四个角点值定义的双线性分布进行重要性采样 |
| **应用场景** | 纹理采样 (texture sampling)、纹理重要性采样、光源图 (light map) 上按亮度采样 |

---

## 2. 数学推导

### 2.1 双线性分布

给定四个角点值 $v_0, v_1, v_2, v_3$（分别对应 $(0,0), (1,0), (0,1), (1,1)$ 处的权重），双线性插值函数为：

$$
B(p_0, p_1) = (1-p_0)(1-p_1)v_0 + p_0(1-p_1)v_1 + (1-p_0)p_1 v_2 + p_0 p_1 v_3
$$

归一化后的 PDF：

$$
f(p_0, p_1) = \frac{4 \cdot B(p_0, p_1)}{v_0 + v_1 + v_2 + v_3}
$$

### 2.2 分解采样

采用条件分布 (conditional distribution) 方法：

**步骤 1：沿 $v$ 方向的边缘分布**

计算 $u=0$ 和 $u=1$ 两条边的平均值：

$$
\bar{v}_0 = v_0 + v_1, \quad \bar{v}_1 = v_2 + v_3
$$

沿 $v$ 方向的边缘分布为从 $\bar{v}_0$ 到 $\bar{v}_1$ 的线性分布。

$$
p_1 = \text{SampleLinear}(u_1, \bar{v}_0, \bar{v}_1)
$$

**步骤 2：给定 $p_1$ 后沿 $u$ 方向的条件分布**

在 $p_1$ 处，$u$ 方向的两个端点值为：

$$
\text{left} = \text{lerp}(p_1, v_0, v_2), \quad \text{right} = \text{lerp}(p_1, v_1, v_3)
$$

$$
p_0 = \text{SampleLinear}(u_0, \text{left}, \text{right})
$$

## 3. 完整代码与逐行注释

```cpp
// 首先在 v 维度采样。计算 u=0 和 u=1 两条边的
// 平均值端点
float v0 = v[0] + v[1], v1 = v[2] + v[3];
// v0 = v[0]+v[1]：底边两端之和（v 方向的边缘分布在 p1=0 处的值）
// v1 = v[2]+v[3]：顶边两端之和（v 方向的边缘分布在 p1=1 处的值）

// 沿该线性分布采样
p[1] = SampleLinear(u[1], v0, v1);
// 使用线性插值采样获得 p[1]（v 方向的坐标）
// SampleLinear 根据端点值 v0, v1 的比例进行重要性采样

// 现在在 u 方向上，从采样的 v 位置处的两个端点值采样
p[0] = SampleLinear(u[0],
                    lerp(p[1], v[0], v[2]),   // 在 p1 处左侧（u=0）的插值值
                    lerp(p[1], v[1], v[3]));   // 在 p1 处右侧（u=1）的插值值
// lerp(p1, v[0], v[2]) = (1-p1)*v[0] + p1*v[2]：u=0 边上在 p1 处的值
// lerp(p1, v[1], v[3]) = (1-p1)*v[1] + p1*v[3]：u=1 边上在 p1 处的值

return p;
// 返回二维采样点 p = (p[0], p[1]) ∈ [0,1]²
```

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u[0]` | `float` | $[0, 1]$ | 均匀随机变量，用于 u 方向采样 |
| `u[1]` | `float` | $[0, 1]$ | 均匀随机变量，用于 v 方向采样 |
| `v[0]` | `float` | $[0, +\infty)$ | 角点 $(0,0)$ 处的权重 |
| `v[1]` | `float` | $[0, +\infty)$ | 角点 $(1,0)$ 处的权重 |
| `v[2]` | `float` | $[0, +\infty)$ | 角点 $(0,1)$ 处的权重 |
| `v[3]` | `float` | $[0, +\infty)$ | 角点 $(1,1)$ 处的权重 |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| `p[0]` | `float` | 采样点的 u 坐标 $\in [0,1]$ |
| `p[1]` | `float` | 采样点的 v 坐标 $\in [0,1]$ |

---

## 5. 应用场景

- **纹理重要性采样**：在纹素 (texel) 级别对纹理亮度进行重要性采样，减少蒙特卡洛方差
- **光源纹理采样**：发光纹理上按亮度分布采样发光点
- **自适应采样**：在图像空间中按误差分布进行自适应采样
- **MipMap 采样的叶节点细化**：在 `SampleMipMap` 找到的纹素内部进一步细化采样位置

## 6. 与相关变换的对比

| 方法 | 维度 | 分布形状 | 依赖 |
|------|------|---------|------|
| **双线性插值采样 (本方法)** | 2D | 四角点定义的双线性 | 依赖 `SampleLinear` |
| **线性插值采样** (`SampleLinear.cpp`) | 1D | 两端点定义的线性 | 基础构建块 |
| **PDF 值计算** (`SampleBilinearInterpolationPDFValue.cpp`) | — | — | 配套 PDF 评估 |
| **MipMap 采样** (`SampleMipMap.cpp`) | 2D | 任意纹理 | 层级结构，粗粒度 |

> **设计模式**：本方法展示了"条件分布分解 (conditional decomposition)"的经典技巧——先沿一个维度从边缘分布采样，再沿另一个维度从给定第一维度结果的条件分布采样。这种方法可以推广到更高维度。
