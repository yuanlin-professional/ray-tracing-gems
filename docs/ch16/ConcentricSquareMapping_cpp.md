# ConcentricSquareMapping.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | Shirley-Chiu 同心正方形映射 (Concentric Square Mapping / Concentric Disk Mapping) |
| **用途** | 将正方形 $[0,1]^2$ 上的均匀采样点映射到半径为 $R$ 的圆盘 (disk) 上，保持面积等比性 (area-preserving) |
| **应用场景** | 圆盘采样、面光源采样 (area light sampling)、景深 (depth of field) 光圈采样、余弦加权半球采样的中间步骤 |

---

## 2. 数学推导

### 2.1 目标

在圆盘上均匀采样，PDF 为：

$$
f(x, y) = \frac{1}{\pi R^2}
$$

### 2.2 传统极坐标法的问题

朴素方法 $r = R\sqrt{u_0}$, $\phi = 2\pi u_1$ 会在圆心附近产生采样点的扭曲和聚集。

### 2.3 同心映射方法

Shirley-Chiu 方法将正方形 $[-1,1]^2$ 划分为上下左右四个区域（以对角线为界），对每个区域分别映射：

将 $u_0, u_1 \in [0,1]$ 映射到 $a, b \in [-1,1]$：

$$
a = 2u_0 - 1, \quad b = 2u_1 - 1
$$

**当 $|a| > |b|$ 时（左右扇区）：**

$$
r = R \cdot a, \quad \phi = \frac{\pi}{4} \cdot \frac{b}{a}
$$

**当 $|a| \le |b|$ 时（上下扇区）：**

$$
r = R \cdot b, \quad \phi = \frac{\pi}{2} - \frac{\pi}{4} \cdot \frac{a}{b}
$$

最终：

$$
X = r\cos\phi, \quad Y = r\sin\phi
$$

这种映射保持正方形到圆盘的面积比例关系，避免了极坐标映射中心的畸变。

## 3. 完整代码与逐行注释

```cpp
// 更多代码说明参见 http://psgraphics.blogspot.com/2011/01/improved-code-for-concentric-map.html

a = 2*u[0] - 1;           // 将 u[0] 从 [0,1] 映射到 [-1,1]，得到正方形 x 坐标
b = 2*u[1] - 1;           // 将 u[1] 从 [0,1] 映射到 [-1,1]，得到正方形 y 坐标

// 特殊情况：避免除以零，由 Greg Ward 指出
if ( b == 0 ) {            // 当 b 为零时
    b = 1;                 // 将 b 设为 1 以避免下方 a/b 的除零错误
}

if (a*a > b*b) {           // 若 |a| > |b|，则处于左右扇区 (sector)
    r = R*a;               // 半径等于 R*a（保留符号信息）
    phi = (M_PI/4)*(b/a);  // 角度在 [-π/4, π/4] 范围内线性插值
} else {                   // 若 |a| <= |b|，则处于上下扇区
    r = R*b;               // 半径等于 R*b
    phi = (M_PI/2) - (M_PI/4)*(a/b);  // 角度在 [π/4, 3π/4] 或对称范围
}

X = r*cos(phi);            // 极坐标转直角坐标：X 分量
Y = r*sin(phi);            // 极坐标转直角坐标：Y 分量
```

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u[0]` | `float` | $[0, 1]$ | 第一个均匀随机变量 |
| `u[1]` | `float` | $[0, 1]$ | 第二个均匀随机变量 |
| `R` | `float` | $(0, +\infty)$ | 圆盘半径 |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| `X` | `float` | 圆盘上采样点的 x 坐标 |
| `Y` | `float` | 圆盘上采样点的 y 坐标 |

---

## 5. 应用场景

- **面光源采样**：对圆盘形面光源进行均匀采样，用于直接光照计算
- **景深模拟**：在圆形光圈 (circular aperture) 上生成采样点，模拟散焦模糊 (defocus blur)
- **余弦加权半球采样**：可将圆盘采样结果投影到半球上，实现 Malley 方法的余弦加权采样
- **各向同性 BRDF 采样**：作为基础映射用于需要圆盘上均匀采样的各种分布

---

## 6. 与相关变换的对比

| 映射方法 | 优点 | 缺点 |
|---------|------|------|
| **同心正方形映射 (本方法)** | 保持面积等比，避免中心聚集；采样点分布均匀；适合低差异序列 (low-discrepancy sequence) | 需要分支判断；需计算三角函数 |
| **极坐标映射** (`PolarMapping.cpp`) | 实现最简单 | 中心附近采样点扭曲，不适合分层采样 (stratified sampling) |
| **八面体同心映射** (`OctahedralConcentricMapping.cpp`) | 直接映射到球面；几乎无分支 | 目标是球面而非圆盘，用途不同 |

## 勘误 (Errata)

**除零风险**：代码仅处理了 `b == 0` 的情况，但当 `a == 0 && b == 0` 时（即输入 $u_0 = 0.5, u_1 = 0.5$，采样点恰好在正方形中心），`a*a > b*b` 判断为 `false`，进入 `else` 分支，此时 `b` 已被修改为 `1`（因为原始 `b == 0`），这使得结果变为 $(0, R)$ 而非原点 $(0, 0)$。更严谨的做法是在开始处添加：

```cpp
if (a == 0 && b == 0) {
    X = 0; Y = 0; return;
}
```
