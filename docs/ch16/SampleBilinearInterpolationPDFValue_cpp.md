## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 双线性插值采样 PDF 值计算 (Bilinear Interpolation Sampling PDF Value) |
| **用途** | 计算双线性插值采样在给定点 $p$ 处的概率密度函数值 |
| **应用场景** | 多重重要性采样 (MIS) 中的 PDF 评估、蒙特卡洛估计器的权重计算 |

## 2. 数学推导

### 2.1 双线性插值 PDF

给定四角点值 $v_0, v_1, v_2, v_3$ 和采样点 $p = (p_0, p_1)$，双线性插值值为：

$$
B(p) = (1-p_0)(1-p_1)v_0 + p_0(1-p_1)v_1 + (1-p_0)p_1 v_2 + p_0 p_1 v_3
$$

归一化 PDF：

$$
f(p) = \frac{B(p)}{\int_0^1\int_0^1 B(p_0, p_1)\, dp_0\, dp_1}
$$

双线性函数在 $[0,1]^2$ 上的积分为：

$$
\int_0^1\int_0^1 B(p)\, dp_0\, dp_1 = \frac{v_0 + v_1 + v_2 + v_3}{4}
$$

因此：

$$
f(p) = \frac{4 \cdot B(p)}{v_0 + v_1 + v_2 + v_3}
$$

## 3. 完整代码与逐行注释

```cpp
return (4 / (v[0] + v[1] + v[2] + v[3])) * Bilerp(p, v);
// PDF = 4 * Bilerp(p, v) / (v[0] + v[1] + v[2] + v[3])
//
// (v[0] + v[1] + v[2] + v[3])：四角点值之和
// 除以 4 得到双线性函数的平均值（即积分值）
// 4 / sum：归一化因子的倒数
// Bilerp(p, v)：在点 p 处的双线性插值值
// 整体返回归一化后的 PDF 值
```

## 5. 应用场景

- **MIS 权重计算**：当使用双线性采样策略与其他采样策略（如均匀采样、BSDF 采样）组合时，需要评估每个样本在各策略下的 PDF
- **蒙特卡洛估计器**：$\hat{I} = \frac{1}{N}\sum_{i=1}^N \frac{f(p_i)}{pdf(p_i)}$，需要 PDF 值来计算贡献权重
- **方差验证**：验证采样算法的正确性——理论 PDF 应与实际采样的直方图匹配

> **实现注意**：`Bilerp(p, v)` 函数计算标准双线性插值 $(1-p_0)(1-p_1)v_0 + p_0(1-p_1)v_1 + (1-p_0)p_1 v_2 + p_0 p_1 v_3$。当四角点值之和为零时（所有角点值均为零），存在除零风险。
