# SampleLinear.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 线性插值采样 (Linear Interpolation Sampling / Sample Linear) |
| **用途** | 在 $[0,1]$ 区间上按端点值 $a$（在 $x=0$ 处）和 $b$（在 $x=1$ 处）定义的线性分布进行重要性采样 |
| **应用场景** | 纹理采样的基础构建块、双线性采样的子步骤、分段线性分布采样 |

## 2. 数学推导

### 2.1 线性 PDF

在 $[0,1]$ 上的线性 PDF 由端点 $a$ 和 $b$ 定义：

$$
f(x) = \frac{2 \cdot \text{lerp}(x, a, b)}{a + b} = \frac{2[(1-x)a + xb]}{a+b}
$$

其中 $\text{lerp}(x, a, b) = (1-x)a + xb$。

### 2.2 CDF

$$
F(x) = \int_0^x f(t)\, dt = \frac{2}{a+b}\left[at - \frac{(a-b)t^2}{2}\right]_0^x = \frac{2ax - (a-b)x^2}{a+b}
$$

简化：

$$
F(x) = \frac{a \cdot 2x + (b-a)x^2}{a+b}
$$

### 2.3 逆 CDF

令 $u = F(x)$，解关于 $x$ 的二次方程：

$$
(b-a)x^2 + 2ax - u(a+b) = 0
$$

使用求根公式并利用 $\text{lerp}(u, a^2, b^2) = (1-u)a^2 + ub^2$ 简化：

$$
x = \frac{a - \sqrt{(1-u)a^2 + ub^2}}{a - b}
$$

**特殊情况**：当 $a = b$ 时，PDF 为常数，$x = u$。

## 3. 完整代码与逐行注释

```cpp
float SampleLinear( float u, float a, float b ) {
// 函数签名：给定均匀随机数 u 和端点值 a, b，返回采样位置

    if (a == b) return u;
    // 特殊情况：当 a == b 时 PDF 为均匀分布，直接返回 u

    return clamp((a - sqrt(lerp(u, a * a, b * b))) / (a - b), 0, 1);
    // 逆 CDF 公式：x = (a - sqrt((1-u)*a² + u*b²)) / (a - b)
    // lerp(u, a*a, b*b) = (1-u)*a² + u*b²：在 a² 和 b² 之间线性插值
    // sqrt(...)：开方
    // (a - sqrt(...)) / (a - b)：完成求根公式
    // clamp(..., 0, 1)：钳制到 [0,1] 以处理浮点误差
}
```

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u` | `float` | $[0, 1]$ | 均匀随机变量 |
| `a` | `float` | $[0, +\infty)$ | $x=0$ 处的分布值（左端点权重） |
| `b` | `float` | $[0, +\infty)$ | $x=1$ 处的分布值（右端点权重） |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| 返回值 | `float` | 采样位置 $x \in [0, 1]$ |

## 5. 应用场景

- **双线性采样的子步骤**：`SampleBilinearInterpolation.cpp` 中沿每个维度调用本函数
- **帐篷函数采样**：`TentFunction.cpp` 中调用本函数来实现帐篷分布的每一半
- **分段线性分布**：将多个线性段拼接，每段使用本函数采样
- **渐变纹理采样**：在亮度从一端到另一端线性变化的纹理上重要性采样
