# TentFunction.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 帐篷函数采样 (Tent Function Sampling) |
| **用途** | 从帐篷函数 (tent function / triangle function) 分布中采样，即在 $[-r, r]$ 上的分段线性对称分布 |
| **应用场景** | 图像重建滤波器 (reconstruction filter)、Mitchell 滤波器的近似、抗锯齿中的像素滤波 |

---

## 2. 数学推导

### 2.1 帐篷函数 PDF

半径为 $r$ 的帐篷函数 PDF：

$$
f(x) = \begin{cases}
\frac{1}{r} - \frac{|x|}{r^2} & \text{if } |x| < r \\
0 & \text{otherwise}
\end{cases}
$$

这是一个以 $x=0$ 为中心、在 $x=0$ 处达到最大值 $1/r$、在 $x=\pm r$ 处降为零的三角形分布。

### 2.2 对称分解

由对称性，可将采样分为两半：
- 以概率 0.5 采样 $[-r, 0]$ 的左半部分
- 以概率 0.5 采样 $[0, r]$ 的右半部分

每一半内，分布从 1 线性衰减到 0，正好是 `SampleLinear(u, 1, 0)` 的形式。

### 2.3 逆 CDF

**若 $u < 0.5$**（左半）：

$$
u' = u / 0.5 = 2u
$$

$$
x = -r \cdot \text{SampleLinear}(u', 1, 0)
$$

**若 $u \ge 0.5$**（右半）：

$$
u' = (u - 0.5) / 0.5 = 2u - 1
$$

$$
x = r \cdot \text{SampleLinear}(u', 1, 0)
$$

---

## 3. 完整代码与逐行注释

```cpp
if (u < 0.5) {
    // 左半部分：采样 [-r, 0]
    u /= 0.5;
    // 将 u 从 [0, 0.5) 重映射到 [0, 1)

    return -r * SampleLinear(u, 1, 0);
    // 从端点 (1, 0) 的线性分布采样，结果在 [0, 1]
    // 乘以 -r 得到 [-r, 0] 范围内的值
    // 注意：SampleLinear(u, 1, 0) 从高端 1 到低端 0 线性衰减

} else {
    // 右半部分：采样 [0, r]
    u = (u - .5) / .5;
    // 将 u 从 [0.5, 1) 重映射到 [0, 1)

    return r * SampleLinear(u, 1, 0);
    // 从端点 (1, 0) 的线性分布采样，结果在 [0, 1]
    // 乘以 r 得到 [0, r] 范围内的值
}
```

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u` | `float` | $[0, 1)$ | 均匀随机变量 |
| `r` | `float` | $(0, +\infty)$ | 帐篷函数的半径（支撑区间为 $[-r, r]$） |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| 返回值 | `float` | 帐篷分布采样值 $\in [-r, r]$ |

---

## 5. 应用场景

- **像素重建滤波器**：在渲染中使用帐篷滤波器进行图像重建，比盒式滤波器 (box filter) 更平滑
- **采样偏移**：在分层采样中，使用帐篷分布而非均匀分布来抖动 (jitter) 采样位置，减少混叠
- **运动模糊**：帐篷滤波器可用于时间域的采样权重
- **抗锯齿**：亚像素偏移的分布，帐篷形给中心更高权重

---

## 6. 与相关变换的对比

| 滤波器/分布 | 形状 | 支撑宽度 | 平滑度 | 特点 |
|------------|------|---------|--------|------|
| **帐篷函数 (本方法)** | 三角形 | $2r$ | $C^0$（连续但不光滑） | 简单高效，优于盒式 |
| **盒式函数 (Box)** | 矩形 | $2r$ | 不连续 | 最简单但有明显锯齿 |
| **高斯函数** (`BoxMullerTransform.cpp`) | 钟形 | $(-\infty, +\infty)$ | $C^\infty$ | 最平滑但无限支撑 |
| **Mitchell-Netravali** | 三次 | $4r$ | $C^1$ | 质量最好但计算较贵 |

> **设计模式**：本函数通过对称分解和复用 `SampleLinear` 来实现帐篷采样，展示了如何将复杂分布分解为简单分布的组合——先离散选择（左/右），再在选中的半段内连续采样。
