# PolarMapping.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 极坐标映射 / 圆盘均匀采样 (Polar Mapping / Uniform Disk Sampling) |
| **用途** | 使用极坐标 (polar coordinates) 方法在半径为 $R$ 的圆盘 (disk) 上均匀采样 |
| **应用场景** | 面光源采样、景深光圈采样、圆盘形几何体上的采样 |

---

## 2. 数学推导

### 2.1 圆盘上的均匀 PDF

在面积为 $\pi R^2$ 的圆盘上均匀采样，PDF 为：

$$
f(x, y) = \frac{1}{\pi R^2}
$$

### 2.2 极坐标参数化

在极坐标 $(r, \phi)$ 中，面积元素为 $dA = r\, dr\, d\phi$。

PDF 变为：

$$
f(r, \phi) = \frac{r}{\pi R^2}
$$

### 2.3 边缘分布

**$\phi$ 的分布**：

$$
\phi = 2\pi u_1
$$

**$r$ 的 CDF**：

$$
F(r) = \int_0^r \frac{2r'}{R^2}\, dr' = \frac{r^2}{R^2}
$$

### 2.4 逆 CDF

令 $u_0 = F(r) = r^2/R^2$：

$$
r = R\sqrt{u_0}
$$

### 2.5 直角坐标

$$
x = r\cos\phi = R\sqrt{u_0}\cos(2\pi u_1)
$$

$$
y = r\sin\phi = R\sqrt{u_0}\sin(2\pi u_1)
$$

---

## 3. 完整代码与逐行注释

```cpp
r = R * sqrt(u[0]);        // 半径：r = R * sqrt(u0)
                            // sqrt(u0) 确保面积均匀分布（而非半径均匀分布）
                            // 若直接用 r = R*u0，则中心附近采样点过密

phi = 2*M_PI*u[1];         // 方位角：φ = 2π*u1，在 [0, 2π) 上均匀分布

x = r*cos(phi);            // 极坐标转直角坐标：x = r*cosφ
y = r*sin(phi);            // 极坐标转直角坐标：y = r*sinφ
```

---

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u[0]` | `float` | $[0, 1]$ | 均匀随机变量，控制半径 |
| `u[1]` | `float` | $[0, 1)$ | 均匀随机变量，控制方位角 |
| `R` | `float` | $(0, +\infty)$ | 圆盘半径 |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| `x` | `float` | 圆盘上采样点的 x 坐标 |
| `y` | `float` | 圆盘上采样点的 y 坐标 |

---

## 5. 应用场景

- **景深效果**：在相机的圆形光圈上采样，模拟散焦模糊
- **面光源**：在圆盘形面光源上采样发光点
- **软阴影**：从面光源上的随机点向着色点投射阴影光线
- **粒子生成**：在圆形区域内均匀散布粒子

---

## 6. 与相关变换的对比

| 方法 | 采样质量 | 实现复杂度 | 分层采样友好度 |
|------|---------|-----------|--------------|
| **极坐标映射 (本方法)** | 面积均匀，但中心有扭曲 | 最简单 | 差——分层样本经映射后扭曲 |
| **同心正方形映射** (`ConcentricSquareMapping.cpp`) | 面积均匀，无扭曲 | 中等（需分支判断） | 好——保持分层结构 |
| **拒绝采样** | 面积均匀 | 简单（但有浪费） | 差——被拒绝的样本浪费 |

> **核心区别**：极坐标映射和同心正方形映射都能在圆盘上实现均匀采样，但极坐标映射在中心附近会将分层采样的矩形网格扭曲为狭长的扇形，破坏分层结构。同心正方形映射通过将正方形分为四个扇区来避免这种扭曲，是更推荐的方法。
