# TriangleQuadWarping.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 三角形采样——四边形扭曲法 (Triangle Sampling via Quad Warping / Square-to-Triangle Mapping) |
| **用途** | 将正方形 $[0,1]^2$ 上的均匀采样点连续映射到三角形内部，保持均匀分布 |
| **应用场景** | 三角形面光源采样、三角形网格上的均匀点分布、蒙特卡洛渲染 |

---

## 2. 数学推导

### 2.1 逆 CDF 推导

三角形上均匀采样等价于对重心坐标 $(\alpha, \beta, \gamma)$ 采样，满足：
- $\alpha + \beta + \gamma = 1$
- $\alpha, \beta, \gamma \ge 0$

采用 $(\beta, \gamma)$ 的联合分布。在标准三角形上，联合 PDF 为常数 $f(\beta, \gamma) = 2$。

**$\beta$ 的边缘分布**：

$$
f_\beta(\beta) = \int_0^{1-\beta} 2\, d\gamma = 2(1-\beta), \quad \beta \in [0,1]
$$

CDF：

$$
F_\beta(\beta) = 2\beta - \beta^2 = 1 - (1-\beta)^2
$$

逆 CDF：

$$
\beta = 1 - \sqrt{u_0}
$$

**给定 $\beta$ 后 $\gamma$ 的条件分布**：

$$
f(\gamma | \beta) = \frac{1}{1-\beta}, \quad \gamma \in [0, 1-\beta]
$$

逆 CDF：

$$
\gamma = (1-\beta) \cdot u_1
$$

**剩余坐标**：

$$
\alpha = 1 - \beta - \gamma
$$

---

## 3. 完整代码与逐行注释

```cpp
beta = 1-sqrt(u[0]);            // β = 1 - sqrt(u0)
                                 // 这是 β 的边缘分布的逆 CDF
                                 // 当 u0=0 → β=0，u0=1 → β=0

gamma = (1-beta)*u[1];          // γ = (1-β) * u1
                                 // 给定 β 后，γ 在 [0, 1-β] 上均匀分布
                                 // (1-β) 是该区间的长度

alpha = 1-beta-gamma;           // α = 1 - β - γ
                                 // 由重心坐标的归一化条件得出

P = alpha*P0 + beta*P1 + gamma*P2;
// 利用重心坐标计算三角形内的采样点
// P = α*P0 + β*P1 + γ*P2
```

---

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u[0]` | `float` | $[0, 1]$ | 均匀随机变量，控制 $\beta$（通过 sqrt 变换） |
| `u[1]` | `float` | $[0, 1]$ | 均匀随机变量，控制 $\gamma$（给定 $\beta$ 的条件分布） |
| `P0, P1, P2` | `vec3` | — | 三角形的三个顶点 |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| `alpha, beta, gamma` | `float` | 重心坐标，满足 $\alpha+\beta+\gamma=1$ |
| `P` | `vec3` | 三角形内的均匀采样点 |

---

## 5. 应用场景

- **三角形面光源**：在三角形光源上均匀采样，用于直接光照的蒙特卡洛估计
- **网格表面采样**：在三角形网格上均匀分布点（如生成表面粒子）
- **光子映射**：从三角形光源上均匀发射光子
- **碰撞检测**：在三角形内部随机采样测试点

## 6. 与相关变换的对比

| 方法 | 映射连续性 | 保持分层性 | 计算开销 |
|------|-----------|-----------|---------|
| **四边形扭曲法 (本方法)** | 连续 $C^0$ | 较好——连续映射保持邻近关系 | 一次 `sqrt` + 几次乘法 |
| **四边形翻折法** (`TriangleQuadFlipping.cpp`) | 不连续（对角线处） | 较差——翻折破坏分层 | 一次比较 + 可能的减法 |
| **Heitz 低差异采样** | 连续 | 最优 | 中等 |

> **经典方法**：这是最广泛使用的三角形采样方法，来自 Osada et al. (2002) 和 Turk (1990) 的工作。`sqrt(u[0])` 是关键操作——它将均匀分布转换为三角形边缘附近的密度衰减，确保面积均匀。

> **与翻折法的区别**：扭曲法是连续映射，不会在对角线处产生不连续性，因此对分层采样和低差异序列更友好。缺点是需要一次 `sqrt` 运算。在现代 GPU 上，这个开销可以忽略不计。
