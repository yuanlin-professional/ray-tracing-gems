# OctahedralConcentricMapping.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 八面体同心映射 (Octahedral Concentric Mapping) |
| **用途** | 将正方形 $[0,1]^2$ 上的均匀采样点直接映射到单位球面 (unit sphere)，保持面积等比性 (equal-area) |
| **应用场景** | 均匀球面采样、球面参数化 (spherical parameterization)、环境贴图存储与采样、法线贴图编码 |

---

## 2. 数学推导

### 2.1 基本思想

该映射分为两步：
1. 将正方形映射到八面体 (octahedron) 的表面
2. 将八面体的点归一化投影到球面

利用八面体与球面之间的等面积关系，实现面积保持映射。

### 2.2 映射步骤

**步骤 1**：将 $u \in [0,1]^2$ 映射到 $[-1,1]^2$：

$$
u' = 2u - 1
$$

**步骤 2**：计算八面体参数：

$$
d = 1 - |u'_0| - |u'_1|
$$

$$
r = 1 - |d|
$$

**步骤 3**：计算角度（在第一象限）：

$$
\phi = \begin{cases} 0 & \text{if } r = 0 \\ \frac{\pi}{4}\left(\frac{|u'_1| - |u'_0|}{r} + 1\right) & \text{otherwise} \end{cases}
$$

**步骤 4**：计算球面坐标：

$$
f = r\sqrt{2 - r^2}
$$

$$
x = f \cdot \text{sign}(u'_0) \cdot \cos\phi
$$

$$
y = f \cdot \text{sign}(u'_1) \cdot \sin\phi
$$

$$
z = \text{sign}(d) \cdot (1 - r^2)
$$

PDF 为：

$$
\text{pdf} = \frac{1}{4\pi}
$$

## 3. 完整代码与逐行注释

```cpp
// 计算半径 r（无分支，branchless）
u = 2*u - 1;                           // 将 [0,1]² 映射到 [-1,1]²

d = 1 - (abs(u[0]) + abs(u[1]));       // 八面体到球面的符号距离 (signed distance)
                                         // d > 0 表示上半球，d < 0 表示下半球

r = 1 - abs(d);                        // 同心映射的径向参数
                                         // r = 0 时为极点，r = 1 时为赤道

// 在第一象限计算 phi（除零测试外无分支），
// 使用 sign(u) 将结果映射到正确的象限
phi = (r == 0) ? 0 : (M_PI/4) * ((abs(u[1]) - abs(u[0])) / r + 1);
// 当 r=0 时（极点），phi 无意义，设为 0
// 否则 phi 根据 u[0] 和 u[1] 的相对大小在 [0, π/2] 范围内计算

f = r * sqrt(2 - r*r);                 // 球面投影的径向缩放因子
                                         // 确保映射后点在单位球面上

x = f * sign(u[0]) * cos(phi);         // x 坐标，sign(u[0]) 确定左右半球
y = f * sign(u[1]) * sin(phi);         // y 坐标，sign(u[1]) 确定前后半球
z = sign(d) * (1 - r*r);              // z 坐标，sign(d) 确定上下半球
                                         // z = ±(1-r²)，在极点 z=±1，赤道 z=0

pdf = 1 / (4*M_PI);                    // 均匀球面采样的 PDF = 1/(4π)
```

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u[0]` | `float` | $[0, 1]$ | 均匀随机变量 |
| `u[1]` | `float` | $[0, 1]$ | 均匀随机变量 |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| `x, y, z` | `float` | 单位球面上的采样点坐标，满足 $x^2+y^2+z^2=1$ |
| `pdf` | `float` | 概率密度 $= 1/(4\pi)$ |

---

## 5. 应用场景

- **均匀球面采样**：替代传统经纬度映射，提供对分层采样和低差异序列更友好的参数化
- **环境贴图**：作为八面体参数化的基础，用于存储和采样环境光照
- **球面纹理映射**：在球面上建立均匀的纹理坐标
- **法线贴图编码**：使用八面体映射高效编码和解码法线方向
- **光线方向量化**：在光线引导 (ray guiding) 中用于方向的离散化

## 6. 与相关变换的对比

| 映射方法 | 目标域 | 面积保持 | 连续性 | GPU 友好度 |
|---------|--------|---------|--------|-----------|
| **八面体同心映射 (本方法)** | 球面 | 是 | 是 | 高（几乎无分支） |
| **经纬度映射** (`LatitudeLongitudeMapping.cpp`) | 球面 | 是 | 是 | 中（需要三角函数） |
| **同心正方形映射** (`ConcentricSquareMapping.cpp`) | 圆盘 | 是 | 是 | 中（有分支） |
| **立方体映射 (cube map)** | 球面 | 否 | 面边界不连续 | 高 |

> **优势**：八面体同心映射的主要优势在于几乎无分支 (branchless) 的实现和对低差异序列的良好保持。`sign` 函数在 GPU 上通常是单指令实现，因此整个映射在 GPU 着色器中非常高效。
