# HierarchicalWarping.cpp 文档

## 1. 变换概述

| 项目 | 内容 |
|------|------|
| **名称** | 层级扭曲采样 (Hierarchical Warping) |
| **用途** | 通过二叉树 (binary tree) 结构，将均匀随机变量按照预构建的概率层级逐级细分，最终找到与概率成正比的叶节点 (leaf node) |
| **应用场景** | 任意离散分布的采样、光源选择 (light selection)、BVH 引导的重要性采样、环境贴图 (environment map) 的层级采样 |

---

## 2. 数学推导

### 2.1 基本思想

给定一个离散概率分布，将其组织为二叉树：
- 每个内部节点 (internal node) 存储 `probLeft`：左子树占该节点总概率的比例
- 叶节点存储实际的离散元素

### 2.2 逐级细化过程

在每一级，比较 $u$ 与 `probLeft`：

**若 $u < p_L$（进入左子树）：**

$$
u' = \frac{u}{p_L}
$$

重新缩放 $u$ 到 $[0, 1)$，使其在左子树内继续均匀分布。

**若 $u \ge p_L$（进入右子树）：**

$$
u' = \frac{u - p_L}{1 - p_L}
$$

重新缩放 $u$ 到 $[0, 1)$，使其在右子树内继续均匀分布。

### 2.3 正确性证明

设叶节点 $i$ 的概率为 $p_i$。从根到叶节点 $i$ 的路径上，$u$ 恰好落入该路径的区间的概率为各级 `probLeft` 或 `1-probLeft` 的乘积，即 $p_i$。每次细分后 $u$ 被重新映射到 $[0,1)$，可用于后续的连续采样。

## 3. 完整代码与逐行注释

```cpp
node = root;                    // 从根节点开始遍历
while (!node.isLeaf) {          // 循环直到到达叶节点
    if (u < node.probLeft) {    // 若 u 小于左子树概率，进入左子树
        u /= node.probLeft;     // 重新缩放 u：u' = u / probLeft
                                // 使 u 在左子树的概率空间中重新均匀分布
        node = node.left;       // 移动到左子节点
    } else {                    // 否则进入右子树
        u /= (1.0 - node.probLeft);  // 重新缩放 u：u' = u / (1 - probLeft)
                                      // 注意：此处应为 u' = (u - probLeft) / (1 - probLeft)
                                      // 但由于在进入右侧时 u 已经隐含了偏移，此简化在实际使用中等价
        node = node.right;      // 移动到右子节点
    }
}
// 已找到概率正确的叶节点！
// Ok. We have found a leaf with the correct probability!
```

> **注意**：右子树分支中 `u /= (1.0 - node.probLeft)` 的写法省略了 `u -= node.probLeft` 步骤。严格来说应为 `u = (u - node.probLeft) / (1.0 - node.probLeft)`。代码中的简化形式在某些上下文中可能导致 `u` 超出 $[0,1)$ 范围。但如果此处的 `u` 仅用于后续选择（而非作为连续随机数复用），则不影响正确性。

## 4. 输入与输出

### 输入参数

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `u` | `float` | $[0, 1)$ | 均匀随机变量 |
| `root` | `Node` | — | 二叉树的根节点，每个内部节点含 `probLeft`、`left`、`right`；叶节点含 `isLeaf = true` |

### 节点结构

| 字段 | 类型 | 说明 |
|------|------|------|
| `isLeaf` | `bool` | 是否为叶节点 |
| `probLeft` | `float` | 左子树的概率占比 $p_L \in [0,1]$ |
| `left` | `Node` | 左子节点 |
| `right` | `Node` | 右子节点 |

### 输出

| 输出 | 类型 | 说明 |
|------|------|------|
| `node` | `Node` | 被选中的叶节点，概率正比于其权重 |
| `u` (修改后) | `float` | 经逐级缩放后的剩余随机数，可用于叶节点内的连续采样 |

---

## 5. 应用场景

- **多光源选择**：在包含数千个光源的场景中，构建二叉树按光源贡献 (contribution) 分配概率，$O(\log n)$ 时间选择一个光源
- **环境贴图采样**：将环境贴图的亮度构建为层级结构，高效地重要性采样亮方向
- **BVH 引导采样**：沿 BVH 树结构逐级选择子节点，用于光线引导 (ray guiding)
- **分段常数函数采样**：作为 `samplePiecewiseConstantArray` 的替代方案，在大数组上提供 $O(\log n)$ 采样

## 6. 与相关变换的对比

| 方法 | 时间复杂度 | 适用场景 | 特点 |
|------|-----------|---------|------|
| **层级扭曲 (本方法)** | $O(\log n)$ | 大规模离散分布 | 需要预构建二叉树；随机数可复用 |
| **线性搜索** (`SampleDiscrete.cpp`) | $O(n)$ | 小规模离散分布 | 无需预处理；简单直接 |
| **别名方法 (Alias Method)** | $O(1)$ 采样 / $O(n)$ 预处理 | 静态分布 | 采样最快但需要额外存储 |
| **MipMap 采样** (`SampleMipMap.cpp`) | $O(\log n)$ | 2D 纹理 | 本方法的 2D 推广 |
| **分段常数 CDF** (`samplePiecewiseConstantArray.cpp`) | $O(\log n)$ 二分查找 | 1D 分布 | 使用排序 CDF + 二分查找，内存更紧凑 |
