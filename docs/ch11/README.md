# Chapter 11: 嵌套体积材质的自动处理

> **Automatic Handling of Materials in Nested Volumes**

## 概述

本章解决了光线追踪中嵌套体积 (Nested Volumes) 材质的处理问题。当多个透明/半透明物体相互嵌套（如水中的冰块、嵌套的玻璃球）时，光线穿过界面时需要正确确定入射和出射介质的材质属性。本章使用栈结构 (Stack) 来追踪光线当前所处的材质层次。

## 核心思想

使用一个体积材质栈追踪光线穿越的所有体积：
- **进入体积**：将材质压入栈 (Push)，确定入射/出射材质
- **离开体积**：从栈中弹出材质 (Pop)，恢复外层材质
- **奇偶校验**：通过计数同一材质出现次数判断进入/离开
- **最顶层标志**：追踪栈中每种材质的最上层实例
