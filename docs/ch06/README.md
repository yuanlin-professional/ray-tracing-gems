# Chapter 6: 快速鲁棒的自相交避免方法

> **A Fast and Robust Method for Avoiding Self-Intersection**

## 概述

本章介绍了一种基于 IEEE 754 浮点数整数位偏移的光线原点偏移方法，用于在光线追踪中避免自相交 (Self-Intersection) 问题。该方法比传统的固定 epsilon 偏移更加鲁棒，能够在不同尺度下正确工作。

## 核心思想

当光线从表面出发（如反射、折射或阴影光线）时，由于浮点精度有限，光线可能错误地与同一表面再次相交。本章的方法利用 IEEE 754 浮点数的整数表示特性，通过对浮点数的整数位进行偏移来精确控制偏移量，使其与表面位置的精度自适应匹配。
